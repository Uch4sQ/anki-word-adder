# Anki Word Adder - インストール方法

> 🇬🇧 English version: [INSTALL.md](INSTALL.md)

## 必要なもの

1. **Firefox** (最新版推奨)
2. **Anki** (デスクトップ版)
3. **AnkiConnect** アドオン

## セットアップ手順

### 1. AnkiConnect をインストール

1. Ankiを開く
2. **ツール → アドオン → 新たにアドオンを取得**
3. コード `2055492159` を入力してインストール
4. Ankiを再起動

### 2. 拡張機能をインストール

#### 方法A: ドラッグ&ドロップ（推奨）

1. Firefoxを開く
2. `anki-word-adder-v1.2.0.xpi` ファイルをFirefoxウィンドウにドラッグ
3. **「追加」** をクリック
4. 完了！

#### 方法B: about:addons から

1. Firefox で `about:addons` を開く
2. 歯車アイコン ⚙️ → **「ファイルからアドオンをインストール」**
3. `anki-word-adder-v1.2.0.xpi` を選択
4. **「追加」** をクリック

## 使い方

### 基本的な使い方

1. **Ankiを起動**しておく
2. Webページで覚えたい単語を選択
3. 右クリック → **「Ankiに追加」** または `Ctrl+Shift+A`
4. モーダルで翻訳・定義を確認
5. **「追加」** をクリック

### 機能

- **自動翻訳**: 日本語 ⇔ 英語
- **辞書API**: 英単語の発音・定義・例文を自動取得
- **文脈抽出**: 単語を含む文を自動で取得
- **画面キャプチャ**: オプションでスクリーンショット添付
- **キーボードショートカット**: `Ctrl+Shift+A`

### 設定

拡張機能アイコンをクリック → **設定** から以下をカスタマイズ:

- デフォルトデッキ
- デフォルトノートタイプ
- 自動翻訳 ON/OFF
- 辞書API ON/OFF
- 画面キャプチャ ON/OFF

## トラブルシューティング

### ❌ AnkiConnectに接続できません

- Ankiが起動しているか確認
- AnkiConnect アドオン（コード: `2055492159`）がインストールされているか確認
- Ankiを再起動

### 翻訳が取得できない

- インターネット接続を確認
- 設定で「自動翻訳」をオフにして、手動で翻訳を入力

### その他の問題

詳しくは [README.md](README.md) の「トラブルシューティング」セクションを参照してください。
