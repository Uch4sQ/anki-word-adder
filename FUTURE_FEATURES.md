## Issue 1: 画像キャプチャ機能の追加

### 概要
選択範囲周辺のスクリーンショットを自動でAnkiカードに添付する機能

### 機能詳細
- 単語を選択した際、周辺の画像をキャプチャ
- モーダルでプレビュー表示
- Ankiカードに画像を埋め込み

### メリット
- 視覚的な記憶が強化される
- 文脈がより分かりやすくなる
- 特に図やチャートを含むコンテンツで有用

### 技術的アプローチ
- `browser.tabs.captureVisibleTab()` API を使用
- 画像をbase64エンコードしてAnkiに送信
- フィールドに `<img>` タグとして埋め込み

## Issue 2: 統計・履歴機能の追加

### 概要
追加した単語の統計情報と履歴を表示する機能

### 機能詳細

**統計ダッシュボード**
- 今日追加した単語数
- 累計単語数
- 週次/月次グラフ
- 最もよく使うデッキ

**履歴表示**
- 最近追加した単語のリスト（10-20件）
- 追加日時、デッキ、ソースURL
- クリックでAnkiで該当カードを開く

### メリット
- 達成感の可視化でモチベーション向上
- 学習進捗の把握
- 以前追加した単語の確認が簡単に

### 技術的アプローチ
- `browser.storage.local` に履歴を保存
- popup.html を作成してブラウザアクションから表示
- Chart.jsなどでグラフ表示

## Issue 3: 辞書API統合で翻訳品質を向上

### 概要
無料翻訳APIより正確な辞書APIを統合し、単語の定義や例文を取得する

### 候補API

1. **Free Dictionary API** (https://dictionaryapi.dev/)
   - 完全無料、APIキー不要
   - 英語の定義、発音、品詞、例文
   - 音声ファイルのURL提供

2. **WordsAPI** (https://www.wordsapi.com/)
   - 無料プランあり
   - 類義語、反意語、語源も取得可能

3. **Merriam-Webster Dictionary API**
   - 無料（要APIキー）
   - 信頼性の高い辞書データ

### 機能詳細
- 単語選択時、翻訳APIに加えて辞書APIも並行で呼び出し
- モーダルに以下を表示：
  - 発音記号
  - 品詞
  - 定義（複数の意味）
  - 例文
  - 類義語
- ユーザーが選択した定義をAnkiに追加

### メリット
- 翻訳よりも正確な意味の理解
- 語彙学習の質が向上
- ネイティブスピーカー向けの情報が得られる

### 技術的アプローチ
- Free Dictionary API から開始（無料・キー不要）
- 設定画面でAPI選択を可能に
- エラーハンドリング：辞書APIが失敗しても翻訳にフォールバック

## 追加のアイデア

他にも以下のような機能が考えられます：

- **キーボードショートカット**: ブラウザ全体で使えるショートカット（例: Ctrl+Shift+A）
- **複数単語の一括追加**: 選択した複数の単語をまとめて追加
- **Cloze deletion サポート**: 穴埋め形式のカード作成
- **既存カードの検索・更新**: 同じ単語が既にあるか確認して更新
- **タグのカスタマイズ**: ドメイン名や日付などを自動タグ化
- **エクスポート機能**: 追加した単語リストをCSVやJSON形式でエクスポート
